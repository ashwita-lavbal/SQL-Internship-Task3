# SQL-Internship-Task3

## Objective
Learn and practice multi-table SQL operations using joins, subqueries, and advanced query techniques to retrieve and analyze relational database data.

## Topics Covered
- INNER JOIN
- LEFT JOIN
- SELF JOIN
- UNION & UNION ALL
- Correlated Subqueries
- DISTINCT
- GROUP BY
- HAVING
- Conditional Aggregation
- Query Optimization
- CREATE INDEX
- EXPLAIN ANALYZE
- COALESCE()
- Common SQL Join Errors

## Database
MySQL Employees Sample Database

## Exercises Completed
- Employee-Department Mapping using INNER JOIN
- Employees with Manager Information using LEFT JOIN
- Combining Active and Former Employees using UNION ALL
- Retrieving Current Salary using Correlated Subqueries
- Displaying Unique Job Titles using DISTINCT
- Calculating Department Gender Ratios
- Join Performance Optimization
- Understanding and Resolving Common SQL Join Errors
